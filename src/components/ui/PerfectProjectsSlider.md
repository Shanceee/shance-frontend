# PerfectProjectsSlider

Компонент горизонтального слайдера для отображения идеальных проектов пользователя.

## Особенности

- **Автоматическое переключение**: Слайды автоматически переключаются каждые 5 секунд
- **Ручная навигация**: Кнопки влево/вправо для ручного переключения
- **Клавиатурная навигация**: Поддержка стрелок влево/вправо
- **Свайпы**: Поддержка свайпов на мобильных устройствах
- **Контроль автопереключения**: Кнопка паузы/воспроизведения
- **Прогресс-бар**: Визуальный индикатор прогресса автопереключения
- **Доступность**: Полная поддержка ARIA и screen readers

## Использование

```tsx
import { PerfectProjectsSlider } from '@/components/ui';

function ProjectsPage() {
  return (
    <div>
      <PerfectProjectsSlider className="my-8" />
    </div>
  );
}
```

## API

### Props

| Prop        | Тип      | По умолчанию | Описание                  |
| ----------- | -------- | ------------ | ------------------------- |
| `className` | `string` | `''`         | Дополнительные CSS классы |

### Состояния

- **Загрузка**: Отображается при загрузке данных
- **Автопереключение**: Автоматическое переключение слайдов
- **Пауза**: Остановка автопереключения при взаимодействии пользователя

## Взаимодействие

### Клавиатура

- `←` - Предыдущий слайд
- `→` - Следующий слайд

### Мышь

- Клик по кнопкам навигации
- Клик по индикаторам слайдов
- Клик по кнопке паузы/воспроизведения

### Сенсорные устройства

- Свайп влево - следующий слайд
- Свайп вправо - предыдущий слайд

## Стилизация

Компонент использует Tailwind CSS классы и поддерживает кастомные стили через `className` prop. Основные стили:

- Градиентный фон карточек проектов
- Backdrop blur эффекты
- Плавные анимации переходов
- Адаптивный дизайн

## Доступность

- ARIA labels для всех интерактивных элементов
- Focus management для клавиатурной навигации
- Screen reader поддержка
- Контрастные цвета для лучшей читаемости
